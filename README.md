## Prerequisites

Make sure you have installed the following tools:

1. Git (minimal version: 2.39.\*)
2. NodeJS (minimal version v18.13.\*)

## Installation

To install and run the application, please follow the instructions below:

1. Clone the repository

```sh
https://github.com/arifpatanduk/task-management-api.git
```

3. Go to your project directory and run:

```sh
npm install
```

4. Create a new database and setting up the environment file

```sh
PORT = your_server_port

DB_HOST = your_database_host
DB_USERNAME = your_database_username
DB_PASSWORD = your_database_password
DB_DATABASE = your_database_name
```

5. Run the server

```sh
npm install
```
